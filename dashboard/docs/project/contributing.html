üöÄ How to Contribute

Getting Started

  1. Fork the repository at github.com/nirholas/xactions
  2. Clone your fork locally
  3. Create a branch for your feature/fix: git checkout -b feature/your-feature
  4. Make changes and commit with clear messages
  5. Push and open a Pull Request

Development Setup

# Clone your fork
git clone https://github.com/YOUR-USERNAME/xactions.git
cd xactions

# Install dependencies
npm install

# Set up environment
cp .env.example .env

# Initialize database
npx prisma generate
npx prisma db push

# Start development
npm run dev
